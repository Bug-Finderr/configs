## Apps

- [Raycast](https://raycast.com) — Spotlight replacement with clipboard history, snippets, window management
- [Ghostty](https://ghostty.org) — GPU-accelerated terminal emulator
- [IINA](https://iina.io) — modern macOS video player
- [Dropzone 4](https://aptonic.com) — drag-and-drop file actions and shortcuts
- [Rectangle](https://rectangleapp.com) — window snapping and tiling via keyboard
- [Duet](https://duetdisplay.com) — use iOS/Android as a second display
- [Free Download Manager](https://freedownloadmanager.org) — download accelerator with torrent support
- [Amphetamine](https://apps.apple.com/app/amphetamine/id937984704) — keep Mac awake on demand

## Brew formulae

- [bat](https://github.com/sharkdp/bat) — `cat` with syntax highlighting and git integration
- [btop](https://github.com/aristocratos/btop) — system monitor for CPU, mem, disk, network
- [eza](https://github.com/eza-community/eza) — modern `ls` with icons and git status
- [direnv](https://github.com/direnv/direnv) — auto-loads `.env` per project directory
- [fd](https://github.com/sharkdp/fd) — fast, user-friendly `find` alternative
- [fzf](https://github.com/junegunn/fzf) — fuzzy finder for files, history, and more
- [gh](https://cli.github.com) — GitHub CLI
- [git-delta](https://github.com/dandavison/delta) — syntax-highlighted git diffs with line numbers
- [lazygit](https://github.com/jesseduffield/lazygit) — terminal UI for git
- [neovim](https://neovim.io) — hyperextensible vim-based editor
- [oh-my-posh](https://ohmyposh.dev) — cross-shell prompt theme engine
- [ripgrep](https://github.com/BurntSushi/ripgrep) — fast recursive grep
- [tldr](https://github.com/tldr-pages/tldr) — community-maintained command cheat sheets
- [yazi](https://github.com/sxyazi/yazi) — terminal file manager with vim keybinds
- [zoxide](https://github.com/ajeetdsouza/zoxide) — smarter `cd` that learns your directories
- [zsh-autosuggestions](https://github.com/zsh-users/zsh-autosuggestions) — history-based ghost text suggestions
- [zsh-syntax-highlighting](https://github.com/zsh-users/zsh-syntax-highlighting) — colors commands as you type
